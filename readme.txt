 == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `shipping-by-city-for-woocommerce.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the menu name WooCommerce->settings.
4. Now we can see new option call "Advance Shipping Zone".
5. After click on this page we can see configuration form.
6. Now just configure & see the result on checkout page. 

== Frequently Asked Questions ==

= How do I activate the plugin in WordPress? =

To activate this plugin, first, go to Plugins > Add New in your WordPress Admin Dashboard. If you're using the direct install method, search for the plugin by name, click Install Now, and then Activate. Alternatively, for the upload method, click Upload Plugin, select the .zip file from your computer, click Install Now, and then Activate

= How do I access the plugin settings? =

Go to WooCommerce > Settings, find "Advance Shipping Zone," configure the settings, and see the results on the checkout page

= Does this plugin conflict with the Table Rate Shipping plugin? =

Our plugin has been upgraded from 1.0.5 to 1.0.6 and is now compatible with the Table Rate Shipping plugin. We have successfully resolved the conflict with the table rate shipping plugin in the 1.0.6 version.

= What happens if the same city matches one or more shipping methods? =

If the same city matches one or more shipping methods, the system will choose the shipping method with the last specified charge.

= Does this plugin work with class-based shipping conditions, for example, '10 * [qty]'?" =

Our plugin has been upgraded from 1.0.5 to 1.0.6 and is now compatible with the shipping class cost. We have successfully resolved the conflict with the shipping class cost in the 1.0.6 version.

= Does this plugin conflict with the Weight Based Shipping Plugin? =

Our plugin does not conflict with weight-based plugins, but it works based on shipping by city. 

= Does this plugin work with city and delivery zone ? =

Yes, our plugin works with different cities and delivery zones

= Does this plugin work using the ZIP code? =

Our plugin works with the city name, so please use the city name instead of the zip code.
